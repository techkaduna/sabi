class sabi.Translator(in_file: str, out_file: str = 'pidgin-5', ajasa: bool = False)

Bases: object

Context manager for translating words in a file to pidgin saved in another file.

Parameters

in_file

Path to the file to be translated.

out_file

Path to the output file where translated words are to be saved. Will create a file named pidgin-n in the working directory if the argument is not specified - where n is random integer between 1 - 10.

ajasa: bool

Randomly start sentence(s) with popular pidgin slangs if True.

::
with Translator(‘english.txt’, ‘pidgin.txt’, ajasa=False) as translated:
if translated is True:

print(‘Translated!’)
